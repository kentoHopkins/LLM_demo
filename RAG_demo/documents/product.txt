FlowMind — Product Specification
==================================

Product Name: FlowMind
Version: 3.2.1
Released: February 2026
Platform: Web (Chrome, Firefox, Safari), iOS, Android

Overview
--------
FlowMind is NovaMind AI's flagship no-code automation platform. It enables users to build AI-powered workflows without writing a single line of code. Workflows can trigger on events (e.g. a new email, a form submission, a Slack message) and perform actions (e.g. update a database, send a reply, generate a report).

Key Features
------------
- Drag-and-drop workflow builder with 200+ pre-built connectors
- AI Step: a native block that calls an LLM to classify, summarize, extract, or generate content
- Conditional logic: if/else branches, loops, and error handlers
- Version history: every workflow change is saved and can be rolled back
- Live debugger: step through a workflow run in real time to identify issues
- Role-based access control: admin, editor, and viewer roles
- SOC 2 Type II certified

System Requirements
-------------------
- Browser: Chrome 110+, Firefox 110+, Safari 16+

Integrations
------------
FlowMind connects natively to: Slack, Gmail, Outlook, Salesforce, HubSpot, Notion, Airtable, Google Sheets, Shopify, Stripe, QuickBooks, Zendesk, Jira, GitHub, Twilio, and 185+ others. Custom REST API and webhook connectors are available on Growth plan and above.
